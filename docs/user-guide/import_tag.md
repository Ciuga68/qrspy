# Description
Imports tags from a text file

## Endpoint
/qrs/tag

## Usage

```
'''
:param filename: The path and filename of the text or csv file
        :example import_tag(r'c:\\some\\folder\\file.txt')
'''
qrs.import_tag('c:/some/folder/tag.txt')
```

## File format

name | 
-----|
tag1 |
tag2 |


## Returns
```
HTTP Status Code
```
