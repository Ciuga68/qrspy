# Description
Returns the privilages on the app specified

## Endpoint
qrs/app/{appid}/privilages

## Supports filter
True

## Usage
```
"""
:param: appid: The appid to return the privilages for

"""
qrs.get_appprivilages('c517fae1-adbb-4848-b404-1d8a8d3748ad')
```
## Returns
```
HTTP Status Code
```
